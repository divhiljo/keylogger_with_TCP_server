# keylogger_with_TCP_server
Outil de type keylogger qui envoie ces donnée vers un serveur avec un protocole TCP


# 🔐 Security Awareness Project

Ce projet s’inscrit dans une démarche de **sensibilisation à la cybersécurité**.
Il vise à illustrer le fonctionnement de certaines menaces afin de mieux comprendre comment les détecter et s’en protéger.

---

## 🎯 Objectif

* comprendre les mécanismes utilisés par certains logiciels malveillants
* observer les comportements réseau suspects
* sensibiliser aux risques liés à la collecte non autorisée de données

---

## 💡 Concept

Ce projet simule un comportement typique observé dans certaines attaques :
la capture d’informations et leur transmission vers un serveur distant.

L’objectif n’est pas l’exploitation, mais **l’analyse et la compréhension** de ce type de flux afin de renforcer les capacités de détection.

---

## 📡 Axe d’étude

* communication réseau entre un client et un serveur
* identification de flux TCP anormaux
* analyse des schémas d’exfiltration de données

---

## 🛡️ Approche

Le projet est utilisé dans un cadre éducatif pour :

* illustrer des scénarios réels de menaces
* comprendre comment ces activités peuvent être détectées
* encourager la mise en place de bonnes pratiques de sécurité

---

## ⚙️ Technologies

* Next.js
* Tailwind CSS

---

## ⚠️ Avertissement

Ce projet est strictement destiné à un usage **éducatif et expérimental**.
Toute utilisation en dehors d’un cadre légal et éthique est interdite.

Projet en développement / expérimentation.

---

## 🔍 Perspective

L’objectif à long terme est d’améliorer les méthodes de détection et de renforcer la sensibilisation aux cybermenaces.
